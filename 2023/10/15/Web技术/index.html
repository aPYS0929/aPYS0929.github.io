1、HTML技术

1.1、网页结构

      完整的HTML页面：<!DOCTYPE html>：         声明为H5文档
                      <html></html>：           页面根元素
                      <head></head>：           头部元素
                           <meta charset =" ">  定义网页编码格式
                           <title></title>：    文章文档标题
                      <body></body>：           可见的页面内容
                      <h1>：定义大标题 <p>定义段落

2、CSS技术

2.2、CSS简介
CSS 指的是层叠样式表 描述了如何显示HTML元素 可同时控制多张网页的布局。
CSS 由‘选择器’ ‘声明块‘组成。
CSS 有四种采用方式( >表示优先级) 内联样式 > 内部样式 > 导入外部样式文件 > 链接外部样式文件，
2.3 内联样式
特点 最为直接 对 HTML 标签使用 style 属性 来添加样式 只能控制一个 HTML元素的表现。
2.4 内部样式
特点 将CSS写在<head></head>标签内 并且用<style></style>表情进行声明 可控制单个 页面的表现。
2.5 导入外部样式
特点 与 链接外部样式文件类似 语法不同。
在页面的 <head><style></style></head>标签内声明
· a  @import "./other.css"
· b  @import url("./other.css")
2.6 链接外部样式
特点 将CSS写在 单独的 CSS文件中 并且在页面中的<head></head>标签内使用<link rel = "stylesheet" href="导入链接或文件路径">
进行声明 每个html文件 都可以引用CCSS文件 方便维护。
