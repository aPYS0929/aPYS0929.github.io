2023/10/12

第一章 hexo搭建

    安装node.js
    安装git
    安装hexo   命令npm install hexo-cli -g
建站
    创建所需文件
        命令 1· 创建指定文件夹: hexo init <folder>
             2. 进入该文件夹: cd <folder>
             3. 安装文件: npm install
    初始化hexo(进入hexo个人博客)
        命令 hexo s

第二章 将hexo部署至github

   注意: 提前注册好自己的GitHub 并设置SSHkey以及创建所需repository
         以及设置好自己的name和email
    部署命令 1. npm install --save hexo-deployer-git
             2. vim _config.yml 配置自己的github网址
             3. hexo d执行

第三章 hexo常用操作

    命令 1.hexo n(new) 创建新篇章 可在source/_posts中用vim进行操作
